# kit_ecoconception_PO-PM


Ce dépôt contient le code source d'un site web statique d'auto-formation à l'écoconception, conçu spécifiquement pour les équipes produit (Product Managers, Product Owners, UX/UI Designers, Développeurs).

**➡️ [Accéder à l'atelier en ligne](https://france-travail.github.io/kit_ecoconception_PO-PM/)** ---

Il est complémentaire au Référentiel d'écoconception, également disponible sur github https://github.com/France-Travail/referentiel-ecoconception, ainsi que : 
    * au **kit de sensibilisation à la conception écoresponsable** https://github.com/France-Travail/sensibilisation-conception-responsable,
    * à la **formation à l'écoconception destinées aux équipes techniques** https://github.com/France-Travail/formation-ecoconception-dev/.
    * au **starter kit d'éconception destiné aux équipes UX/UI** https://github.com/France-Travail/starter-kit-ecoconception

## À propos

Ce site web est un kit d'atelier "prêt à l'emploi" pour s'acculturer aux concepts de la sobriété numérique. Initialement basé sur des supports de formation internes, il a été rendu générique pour être utilisable par toute organisation.

Il contient deux outils principaux :

1.  **Trois modules d'exercices pratiques :**
    * **Démarrage :** Clarifier les rôles et évaluer la maturité.
    * **Diagnostic & Priorisation :** Apprendre à utiliser les outils d'audit (Green IT Analysis, Console) pour trouver des axes d'amélioration.
    * **Communiquer :** Savoir "traduire" les gains techniques (gCO2e) en arguments concrets (km en voiture, etc.) avec Impact CO2.

2.  **Un Radar de Maturité V2.0 interactif :**
    * Un outil complet (basé sur un tableur d'analyse détaillé) pour permettre aux équipes d'auto-évaluer leur niveau d'application des bonnes pratiques (RGESN, etc.) et de visualiser leurs forces et faiblesses sur un graphique radar.

## Comment l'utiliser ?

Ce site est un simple site web statique (HTML, CSS, JS). Il n'y a pas de base de données ni de backend.

* **En ligne (Recommandé) :** Le site est hébergé avec GitHub Pages et est accessible à l'adresse indiquée ci-dessus.
* **En local :** Clonez ce dépôt (`git clone ...`) puis ouvrez le fichier `index.html` directement dans votre navigateur.

Le parcours de formation commence sur la page d'accueil et vous guide à travers les différents modules.
