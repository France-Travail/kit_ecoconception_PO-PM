Module 2 : Diagnostic & Priorisation

Nous passons de la théorie à la pratique. Dans ce module, vous allez apprendre à utiliser les outils professionnels pour auditer un service numérique existant, identifier les "points rouges" de performance et d'impact, et prioriser les actions correctives.

Exercice 3 : Auditer votre service

Nous allons utiliser deux outils principaux pour collecter des données sur votre service : l'extension Green IT Analysis et la Console Développeur de votre navigateur.

Partie A : Installation et audit Green IT Analysis

Guide d'installation et d'utilisation :

  1. Ouvrez Firefox (ou Chrome) et rendez-vous sur le store des extensions.
  2. Recherchez et installez "Green IT Analysis".
  3. Une fois installée, ouvrez la page web que vous souhaitez analyser.
  4. Cliquez sur l'icône de l'extension dans la barre d'outils de votre navigateur.
  5. Lancez l'analyse et attendez quelques secondes pour obtenir les résultats.

Conseil rapide : Vous pouvez aussi analyser les sites web de vos concurrents pour vous comparer !

Partie B : Analyse de l'onglet Réseau

Guide d'utilisation de la Console Développeur :

  1. Sur la page à analyser, ouvrez les outils de développement (clic droit > "Inspecter" ou touche F12).
  2. Allez dans l'onglet "Réseau" (Network).
  3. Cochez la case "Désactiver le cache" pour simuler une première visite.
  4. Rechargez la page (F5) et observez les résultats en bas de la fenêtre (nombre de requêtes, poids total...).

À vous de jouer : Remplissez votre diagnostic

Utilisez les outils que vous venez de voir pour auditer un site de votre choix (votre site actuel, un concurrent, etc.) et remplissez le tableau de diagnostic suivant.
